# Delegation Templates

## Canonical Template

```markdown
### Delegation
- Target Agent: daily-subagent
- Assigned Skill: [Skill Name] (e.g., meeting-prep)
- Task: [Specific User Intent]
- Context: [Outputs from Context Engineer]
```

## Notion-related

```markdown
### Delegation
- Target Agent: daily-subagent
- Assigned Skill: notion-use
- Task: [User intent tied to Notion page/database workflow]
- Context: [Relevant page URLs, data source IDs, schema notes from context-engineer]
```

## Visualization

```markdown
### Delegation
- Target Agent: daily-subagent
- Assigned Skill: visual-explainer
- Task: [Visual output objective and audience]
- Context: [Source docs, required diagram type, constraints]
```

## Deep Research

```markdown
### Delegation

- Context: [Initial hypothesis, key unknowns, priority sources from context-engineer]
```
